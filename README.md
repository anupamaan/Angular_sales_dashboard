# Angular Sales Dashboard

A responsive Sales Dashboard built using Angular and Chart.js to visualize sales performance through interactive charts and KPI cards.

## Features

* Dashboard KPI Cards

  * Total Sales
  * Total Orders
  * Total Customers

* Interactive Charts

  * Pie Chart for Region-wise Sales
  * Line Chart for Monthly Sales Trends

* Dynamic Data Integration

  * API-driven dashboard
  * Real-time chart updates using Angular Inputs

* Responsive UI Design

  * Mobile-friendly layout
  * Clean and modern dashboard interface

## Technologies Used

* Angular
* TypeScript
* Chart.js
* HTML5
* CSS3

## Installation

1. Clone the repository

git clone <repository-url>

2. Navigate to the project directory

cd angular-sales-dashboard

3. Install dependencies

npm install

4. Run the application

ng serve

5. Open in browser

http://localhost:4200
